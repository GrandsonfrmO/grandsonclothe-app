# ÉTAPE 11 : DÉMARRAGE RAPIDE ⚡

## 5 minutes pour commencer

### Étape 1: Accéder à l'admin (30 secondes)
```
http://localhost:3000/admin
```

### Étape 2: Voir les produits (30 secondes)
1. Cliquez sur "Produits" dans la sidebar
2. Vous voyez la liste des produits

### Étape 3: Ajouter un produit (2 minutes)
1. Cliquez sur "Nouveau produit"
2. Remplissez:
   - Nom: "Mon produit"
   - Prix: "29.99"
   - Stock: "50"
3. Cliquez "Créer"

### Étape 4: Voir les commandes (1 minute)
1. Cliquez sur "Commandes" dans la sidebar
2. Vous voyez la liste des commandes

### Étape 5: Changer le statut d'une commande (1 minute)
1. Cliquez sur "Détails" pour une commande
2. Sélectionnez un nouveau statut
3. Le statut est mis à jour automatiquement

## Raccourcis

### Ajouter un produit
1. `/admin/products`
2. Cliquez "Nouveau produit"
3. Remplissez le formulaire
4. Cliquez "Créer"

### Modifier un produit
1. `/admin/products`
2. Cliquez sur l'icône "Modifier"
3. Modifiez les champs
4. Cliquez "Modifier"

### Supprimer un produit
1. `/admin/products`
2. Cliquez sur l'icône "Supprimer"
3. Confirmez la suppression

### Changer le statut d'une commande
1. `/admin/orders`
2. Cliquez sur "Détails"
3. Sélectionnez le nouveau statut
4. Le statut est mis à jour

### Voir l'historique d'un client
1. `/admin/customers`
2. Cliquez sur "Historique"
3. Consultez les commandes

---

## Statuts des commandes

- 🟡 **pending** - En attente
- 🔵 **processing** - En traitement
- 🟣 **shipped** - Expédié
- 🟢 **delivered** - Livré
- 🔴 **cancelled** - Annulé

## Conseils rapides

### Produits
- Mettez à jour le stock régulièrement
- Utilisez des catégories cohérentes
- Ajoutez des images de qualité

### Commandes
- Changez le statut rapidement
- Vérifiez l'adresse de livraison
- Confirmez le numéro de téléphone

### Clients
- Consultez l'historique régulièrement
- Identifiez les clients VIP
- Suivez les tendances d'achat

### Statistiques
- Vérifiez le revenu quotidien
- Analysez les statuts des commandes
- Identifiez les produits populaires
